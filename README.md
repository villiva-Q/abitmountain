# abitmountain

比特币挖矿成本收益计算器（Web 单页 Version 1）

## 当前功能
- 矿箱组合：支持 `1MW / 2.4MW` 多规格组合与数量输入
- 多币种输入：`USD / AUD / RMB`
- 自动汇率：`USD/AUD`、`USD/RMB`
- 自动 BTC 价格拉取
- 收益模式：
  - **网络参数估算（默认）**
  - **Hashprice 模式（备用）**
- 网络参数字段：
  - 全网算力 `EH/s`
  - 网络难度 `Difficulty`
  - 区块奖励
  - 平均每区块手续费
- 结果输出：
  - 每日毛收入
  - 每日电费
  - 每日净收益
  - 静态回本周期
  - 预计日产 BTC
  - 每度电产出收入
  - 盈亏平衡电价
  - 容量校验 / 装载率 / 最大部署台数
- 本地保存方案：浏览器 localStorage
- 页面内置 **估算依据说明**

## 估算逻辑
默认使用“网络参数估算”模式：

- 矿机算力占比 = 自身总算力 / 全网总算力
- 日产 BTC = `144 × (区块奖励 + 平均手续费) × 算力占比`
- 日毛收入 = 日产 BTC × BTC 价格
- 日净收益 = 日毛收入 - 电费

> 当前为实时参数下的静态估算模型，不代表未来实际收益。

## 实时数据
当前页面会尝试自动拉取：
- BTC 价格
- USD/AUD 汇率
- USD/RMB 汇率
- 比特币网络难度
- 比特币区块高度（用于显示网络数据更新时间）

如果外部接口失败，可改为手动输入参数继续计算。
